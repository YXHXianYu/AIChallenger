关于set_mode.bat说明:
*参数至少为5个(4个数字)，否则会溢出
1.编辑方式
	右键bat -> 编辑
2.内容说明
	参数1: 为程序名(不变)
	参数2: 测试模式
		=1 单人模式
		=2 双人模式
		=3 双人对抗
		=4 AI模式
		=5 AI对抗AI
	参数3: 玩家1扮演角色
		=1 弓箭手
		=2 猎人
		=3 史莱姆
	参数4(仅支持存在双角色的模式): 玩家2扮演角色
		=1 弓箭手
		=2 猎人
		=3 史莱姆
	参数5(仅支持存在怪物的模式): 怪物难度选择
		=0 简单模式(波数少)
		=1 普通难度(怪物会攻击视野范围内的敌方角色)
		=2 困难难度(怪物智力视野增大)
		=3 智慧难度(怪物智力非常高，会学习)
		=4 绞肉难度
3.备注
	"启动bat后会自动开启测试"

4.绞肉
	参数2 = 6
	*默认为绞肉难度
	*其他参数目前不提供设置方式

5.AI训练：
	参数2 = 7
	参数3 = AI扮演角色 (若参数3为0，则无"主要AI"进行训练)
	参数4 = 训练次数
	参数5 = 怪物难度
